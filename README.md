# Homebrew Tap for ProxyForge

Homebrew tap for [ProxyForge](https://github.com/doubleniki/proxy-forge) - a secure SOCKS5 proxy client for macOS.

## Installation

```bash
brew tap doubleniki/proxy-forge
brew install --cask proxy-forge
```

Or install directly:

```bash
brew install --cask doubleniki/proxy-forge/proxy-forge
```

## Updating

```bash
brew upgrade --cask proxy-forge
```

## Uninstallation

```bash
brew uninstall --cask proxy-forge
```

## Requirements

- macOS Big Sur (11.0) or later

## About ProxyForge

ProxyForge is a next-generation macOS client for Shadowsocks protocol, featuring:

- SOCKS5 and HTTP proxy support
- PAC (Proxy Auto-Configuration) with GFW list
- Multiple proxy modes: Auto, Global, Manual
- SIP003 plugin support (v2ray-plugin, kcptun, simple-obfs)
- QR code scanning and sharing
- Native macOS integration

For more information, visit the [main repository](https://github.com/doubleniki/proxy-forge).
